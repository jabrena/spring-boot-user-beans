# Spring boot User Beans

A project to learn to understand better the Beans that you use in your projects.

![](./docs/user-beans.png)

## How to run in local

```
mvn spring-boot:run
http://localhost:8080/beansviz
http://localhost:8080
```

## Spring Boot CLI

```
sdk install springboot
spring init -d=web,devtools --build=maven --force ./
```
